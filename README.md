# Jingze Tian — Personal Academic Website

Personal academic website of [Jingze Tian 田景泽](https://jingzetian.github.io), HCI researcher at City University of Hong Kong.

**Live site:** https://jingzetian.github.io

Built with [Hugo](https://gohugo.io/) and [Wowchemy](https://wowchemy.com/).

## Local development

```bash
hugo server
```

## Deploy

The site is deployed automatically to GitHub Pages on push to `main` via GitHub Actions.
