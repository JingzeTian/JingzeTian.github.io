“TIMER” 是一款智能戒指，用户可以随时随地记录自己的活动。与其他商业可穿戴设备（如智能手环）类似，它与手机端应用配合使用。

工具：
- 建模与渲染：**Blender**
- Logo 设计：**Adobe Illustrator (AI)**
- UI 设计：**Figma**
