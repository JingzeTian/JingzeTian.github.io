# 简介

在[牛亚峰教授](https://me.seu.edu.cn/nyf_31777/list.htm)指导下，我的本科毕业设计开发了一套基于 Fitts 定律的人因实验平台，并设计了三因素被试内实验，招募 20 名被试使用该平台完成实验。

我的职责包括：
- 使用 Unity 与 Tobii SDK for Unity 搭建实验平台。
- 为平台加入以下功能：
  - 实验流程配置
  - 数据采集
  - 数据导出
  - 用户数据记录

目前，该实验平台（含后续迭代与优化）已支撑完成两篇本科毕业论文、两篇硕士学位论文，以及两篇期刊论文。

此外，我还在[第二届中国设计人因学会暨第 21 届国际管理工效学会议](https://mp.weixin.qq.com/s/dsli6iNf89vbVmiW62uutA)上汇报了该项目。
